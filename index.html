The Commander Cube

Version 0.0.0

About

The commander cube is a collection of five mono-colored Commander (EDH) decks with the idea that you should be able to shuffle all the non-basics from the different decks together to form a cohesive cube for drafting.

The Core Principles

The Commander Cube tries its best to stick to these guidelines:

  • Each commander deck must contain exactly 28 basic lands. This leaves 72 cards for each respective color, including artifacts and non-basic lands, making for a cube of 360 cards.
  • There should be a power level balance between the Commander lists and similarly in the cube's dominant archetypes - which need not be strictly along color lines.
  • Decks should stick close to a rarity breakdown mirroring that of a standard booster pack. Note that this is only a guideline and a secondary concern to balanced power.
    • 5 rares/mythic rares
    • 15 uncommons
    • 50 commons
    • 2 flex slots
